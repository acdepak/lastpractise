Creating a Repository

This is on linux. Be at the desired directory and start your task
Create a new Repository on the github website.(Make sure you have an account;)
touch filename Or you can make a folder using mkdir foldername
code . Or gedit filename
git add filename
git commit -m "write a message(e.g. First commit)"
git remote add origin git@github.com:username/newRepo
git push -u origin main

Uploading into a existing Repository

Make one or more files or folder, then do the following
git add .
git commit -m " added some files or folders"
git branch -m main
git push -u origin main
